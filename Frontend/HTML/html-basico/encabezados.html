Los encabezados

Hay seis elementos de encabezado en HTML: h1 a h6. Se utilizan para mostrar la importancia de las secciones de tu página web, siendo h1 el más importante y h6 el menos importante.

Un elemento h1 es el encabezado principal de una página web y solo se debe usar uno por página. Los elementos h2 (o h3,h4,h5,hasta h6) representan subtítulos.

Esto seria un encabezado principal de tipo h1: <h1>Contenido</h1>

Esto seria un subtitulo de nivel h2: <h2>Contenido</h2>

Esto seria un subtitulo de nivel h3: <h3>Contenido</h3>

Esto seria un subtitulo de nivel h4: <h4>Contenido</h4>

Esto seria un subtitulo de nivel h5: <h5>Contenido</h5>

Esto seria un subtitulo de nivel h6 (el mas pequeño y menos importante de los encabezados): <h6>Contenido</h6>
